    \subsection{1. Basic terms}\label{basic-terms}

\(n\) observations, each with \(d\) features/predictors. Some
observations belong to class C; some do not.

\textbf{decision function}

\(f(x)>0\), if \(x \in class C\)

\(f(x) \leq0\). if \(x \notin class C\)

aka: predictor function or discriminant function

\textbf{decision boundary}

the boundary chosen by classifier to separate items in the class from
those not.

\(\{x \in \mathbb{R}^d: f(x) = 0\}\)

This set is a \((d-1)\)-dimentional surface in \(\mathbb{R}^d\). Hence
it is also called an isosurface of \(f\) for the isovalue 0.

    \subsection{2. Linear Classifier}\label{linear-classifier}

The decision boundary is a line/plane.

\textbf{Decision function}

\(f(x) = w\cdot x + \alpha\)

\textbf{Decision boundary}

\(H = \{x: w\cdot x = -\alpha\}\)

\(H\) is called a hyperplane ( A line in 2D, a plane in 3D)

\textbf{Theorem}

Let \(x\),\(y\) be 2 points that lie on \(H\). Then \(w\cdot (y-x)=0\)

\(w\) is called the \emph{normal vector} of \(H\), since \(w\) is normal
(perpendicular) to \(H\)

\textbf{signed distance} from \(x\) to \(H\)

\(\frac{1}{|w|}(w\cdot x + \alpha)\)

i.e. positive on one side of \(H\); negative on other side.

Moreover, the distance from \(H\) to the origin is \(\alpha\). Hence
\(\alpha=0\) if and only if \(H\) passes through origin.

\textbf{Definition}

The input data is \textbf{linearly separable} if there exists a
hyperplane that separates all the sample points in class C from all the
points NOT in class C.

    \subsubsection{2.1 Centroid Method}\label{centroid-method}

Compute mean \(\mu_C\) of all vectors in class C and mean \(\mu_X\) of
all vectors NOT in C.

\textbf{decision function}

\(f(x) = (\mu_C -\mu_X)\cdot x - (\mu_C - \mu_X)\frac{\mu_C+\mu_X}{2}\)

\textbf{decision boundary}

the hyperplane that perpendicular bisects line segment w/endpoints
\(\mu_C\),\(\mu_X\)

    \subsubsection{2.2 Perceptron Algorithm}\label{perceptron-algorithm}

Consider \(n\) sample points \(X_1, X_2, ..., X_n\).

For each sample point, the label
\(y_i = \begin{cases}1 & \quad\text{if }X_i \in C \\ -1 & \quad\text{if }X_i\notin C \end{cases}\)

For simplicity, consider only decision boundaries that pass through the
origin, that is, \(\alpha = 0\)

If it doesn't pass through origin, add a fictious dimension:

\(f(x) = w'\cdot x' = \begin{bmatrix}w_1&w_2&\cdots&w_d&\alpha\end{bmatrix}\cdot\begin{bmatrix}x_1\\x_2\\\vdots\\x_d\\1\end{bmatrix}\)

Now we have sample points in \(\mathbb{R}^{d+1}\), all lying on
hyperplane \(x_{d+1}=1\).

\textbf{Goal}

Find weights \(w\) such that

\(X_i\cdot w \geq 0\), if \(y_i=1\), and

\(X_i\cdot w \leq 0\), if \(y_i = -1\).

Equivalently: \(y_iX_i\cdot w\geq0\). This is called a
\emph{constraint}.

\textbf{Idea}

Define a \emph{risk function} \(R\), positive if some constrains are
violated. Then we use optimization to choose \(w\) that minimizes \(R\).

\textbf{Loss function}

\(L(z,y_i) = \begin{cases}0 & \quad\text{if }y_iz\geq0 \\ -y_iz & \quad\text{otherwise}\end{cases}\)

\textbf{Risk function}

\(R(w) = \Sigma_{i=1}^nL(X_i\cdot w, y_i) = \Sigma_{i\in V}-y_iX_i\cdot w\)

Where \(V\) is the set of indices \(i\) for which \(y_iX_i\cdot w< 0\)

if \(w\) classifies all \(X_1, ..., X_n\) correctly, then \(R(w)=0\).

Otherwise, \(R(w)\) is positive, and we want to find a better value of
w.

\textbf{Modified goal}

Find \(w\) that minimizes \(R(w)\)

\textbf{Optimization algorithm} (not perceptron yet)

Given a starting point \(w\) (good choice: any \(y_iX_i\))

While \(R(w)>0\): \(w\gets w + \epsilon \cdot (-\triangledown R(w))\)

return \(w\)

\begin{itemize}
\tightlist
\item
  \(\triangledown R(w)\), is the direction of the steepest ascent. Hence
  take a step in the opposite direction,
  \[-\triangledown R(w) = \Sigma_{i \in V}y_iX_i\]
\item
  \(\epsilon >0\) is the \emph{step size} aka \emph{learning rate}
\item
  Problem: Slow! Each step takes \(O(nd)\) times
\end{itemize}

\textbf{Optimization algorithm 2} aka \textbf{Perceptron Algorithm}

Given a starting point \(w\)

While some \(y_iX_i\cdot w <0\): \(w \gets w + \epsilon \cdot y_iX_i\)

return \(w\)

\begin{itemize}
\tightlist
\item
  each step, only pick one misclassified \(X_i\), do gradient descent on
  loss function \(L(X_i\cdot w, y_i)\)
\item
  each step takes \(O(d)\) times. (Not counting the time to search for a
  misclassified \(X_i\))
\end{itemize}

    \subsubsection{2.3 Maximum Margin Classifiers (hard margin
SVM)}\label{maximum-margin-classifiers-hard-margin-svm}

The \textbf{margin} of a linear classifier is the distance from the
decision boundary to the nearest sample point.

\textbf{Goal}

Make the margin as wide as possible

\(margin = min_i \frac{1}{|w|}|w\cdot X_i+\alpha|\)

\textbf{Constraints}

\(y_i(w\cdot X_i + \alpha)\geq 1\) for \(i \in [1,n]\)

Notice that the right-hand side is a \(1\), rather than \(0\) as it was
for the perceptron algorithm.

Intuition: not allow points falling on the boundary, that is, margin is
at least larger than 0.

\textbf{Optimization problem}

Find \(w\) and \(\alpha\) that maximize margin
\(max_{w, \alpha}min_i \frac{1}{|w|}|w\cdot X_i+\alpha| = max_{w, \alpha}\frac{1}{|w|} = min_{w, \alpha}|w|^2\)

Subject to \(y_i(X_i\cdot w + \alpha)\geq 1\) for all \(i \in [1,n]\)

\begin{itemize}
\tightlist
\item
  It is a quadratic program in \(d+1\) dimensions and \(n\) contraints.
\item
  It has one unique solution / no solution.
\item
  Use \(|w|^2\) instead of \(|w|\) because \(|w|\) is not smooth at
  zero, while \(|w|^2\) is smooth everywhere.
\item
  Only applied to those linear seperatable
\item
  Sensitive to outliers.
\end{itemize}

    \subsubsection{2.4 Soft-Margin Support Vector Machines
(SVMs)}\label{soft-margin-support-vector-machines-svms}

\textbf{Idea}

Allow some points to violate the margin, with slack varibles

\textbf{Constrains}

\(y_i(X_i\cdot w +\alpha) >= 1-\xi _i\) and

\(\xi_i \geq 0\) for \(i \in [1,n]\)

\begin{itemize}
\tightlist
\item
  sample points that don't violate the margin all have \(\xi_i = 0\)
\item
  Point \(i\) has nonzero \(\xi_i\) if and only if it violates the
  margin.
\item
  \(\frac{\xi_i}{|w|}\) is how much the point \(i\) violets the margin.
\end{itemize}

\textbf{Optimization problem}

Find \(w\), \(\alpha\) and \(\xi_i\) that minimize
\(|w|^2 + C\Sigma_{i=1}^n\xi_i\)

Subject to \(y_i(X_i\cdot w +\alpha) >= 1-\xi _i\)

\(\xi_i \geq 0\) for \(i \in [1,n]\)

\begin{itemize}
\tightlist
\item
  It is a quadratic program in \(d+n+1\) dimensions and \(2n\)
  constrains.
\item
  \(C>0\) is a scalar regularization hyperparameter. Larger \(C\)
  increase the penalty of \(\xi_i \neq 0\)
\item
  If \(C\) is infinite, we're back to a hard-margin SVM.
\end{itemize}
